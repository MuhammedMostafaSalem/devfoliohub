# DevFolioHub

DevFolioHub is a platform for developers to showcase their projects, build their portfolio, and deploy frontend and backend applications with custom URLs.

## Features
- Developer Portfolio Pages
- Project Showcase
- Frontend & Backend Deployments
- Custom Project URLs
- Dashboard for Managing Projects
- Deployment Management

## Tech Stack
- Node.js
- Express.js
- MongoDB
- React / Next.js
- Tailwind CSS

## Project Structure
This project will be built using a modular architecture with separate modules for:
- Authentication
- Users & Profiles
- Projects
- Deployments

Project in development.
